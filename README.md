# Kieyara Portfolio

A modern, responsive personal portfolio website showcasing my journey as a Software Developer, AI Builder, and Bachelor of Science in Information Technology student.

## About

This portfolio highlights my technical skills, projects, achievements, and experience in software development, artificial intelligence, and web technologies. It is designed with a modern UI, smooth animations, and responsive layouts to provide an engaging user experience.

## Features

- Modern and responsive design
- Interactive user interface
- Animated landing page
- About Me section
- Skills showcase
- Project portfolio
- Education and experience timeline
- Contact form
- GitHub and LinkedIn integration
- Downloadable resume
